PRESSIA LAUNDRY MANAGEMENT - TESTING

HOW TO RUN:
1. Install Node.js from https://nodejs.org/ (download LTS version)
2. Double-click START_APP.bat
3. Wait for the desktop app to open
4. Test all features!

WHAT TO TEST:
- Create new orders
- Track orders with due dates
- Print receipts
- Switch languages (English/French)
- Add expenses
- View dashboard
- Export/import data

IF IT DOESN'T WORK:
1. Make sure Node.js is installed
2. Try running: npm install
3. Then try: npm run electron-dev

The app will open as a desktop window (not in browser).
All data is stored locally on your computer.

Enjoy testing!
